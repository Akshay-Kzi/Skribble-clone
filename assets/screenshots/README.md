# Screenshots

Place your game screenshots here. Recommended filenames:

- game-interface.png - Main game view with canvas and chat
- drawing-tools.png - Close-up of the drawing tools (pencil, bucket, eraser)
- leaderboard.png - Game over screen with podium-style leaderboard
- room-creation.png - Room creation/join interface
- multiplayer.png - Multiple players playing together

Recommended image specs:
- Width: 1200px or wider
- Format: PNG (for better quality) or JPG
- Keep file sizes reasonable (under 1MB each)
